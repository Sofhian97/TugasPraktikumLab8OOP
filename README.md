Pertama masukkan data dengan pilihan 'T'. masukkan nama, nim, dan nilai. 
Untuk melihat data gunakan pilihan 'L'. maka data yang sudah dimasukkan tadi akan muncul pada sebuah tabel. 
untuk mengubah data perlu memasukkan nama untuk dicari apakah data dengan nama tersebut ada atau tidak ada. jika ada maka mengubah data bisa dilakukan, jika tidak ada maka ulangi perintah ubah 'U' dengan memasukkan nama yang benar. 
'H' digunakan untuk menghapus data. cukup dengan memasukkan nama nya saja maka data di dalamnya yaitu nim dan nilai akan terhapus. untuk meyakinkan bisa gunakan 'L' lagi agar bisa terlihat apakah data terhapus dan berubah atau belum.
